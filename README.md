# HBM Panic Emergency Injection

A Minecraft Forge mod for version 1.7.10.

## Development Setup

1.  Ensure you have Java 8 installed.
2.  Clone the repository.
3.  Run `./gradlew setupDevWorkspace` to set up the environment.
4.  Run `./gradlew eclipse` (or `idea`) to generate IDE project files.
5.  (Optional) Add required library jars to the `libs/` folder if they are not automatically downloaded.

## Building

Run `./gradlew build` to create the mod jar in `build/libs/`.
